### 项目介绍
***
该项目是一款在线教育平台，包含后端管理平台以及前端、移动端平台，基本上包括了此类平台大部分功能。  
同时，由于并未使用任何其他后台框架，并没有冗余的功能，项目启动速度快，体验极佳，更多功能正在开发中。

### 快速开始
***
下载项目
```shell
git clone linuxterminator/qingshan

cd qingshan
```
启动项目
```shell
mvn install

mvn spring-boot:run
```
项目打包
```shell
mvn package
```
